块级元素,独占一行 线性布局 方向:竖直方向

胆小如鼠 inline （行内元素）: 不会独占一行. 不能更改 height，width的值，大小由内容撑开. 可以使用padding上下左右都有效，margin只有left和right有效，但是top和bottom就不行. block （块级元素）: 独占一行，在不设置自己的宽度的情况下， 默认填满父级元素的宽度. 能够改变 height，width的值. 可以设置padding，margin的各个属性值，top，left，bottom，right都能够产生边距效果. inline-block （融合行内于块级）: 结合了inline与block的一些特点， 不会独占一行. 能够改变 height，width的值. 设置padding，margin的各个属性值，top，left，bottom，right都能够产生边距效果 用通俗的话讲，就是不独占一行的块级元素。
